# CoachFlow – Plateforme de vente de guide

## Présentation
CoachFlow est une application web de coaching sportif permettant :
- aux coachs de vendre des guides d’entraînement numériques
- aux élèves d’acheter ces guides et de réserver des séances de coaching privées.

Le projet met l’accent sur une logique métier robuste, la sécurité applicative et la gestion de concurrence.
